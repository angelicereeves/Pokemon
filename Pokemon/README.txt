# Pokémon Search App
Author: Angelica Reeves

This is a simple web app that allows users to search for any Pokémon by name or ID using
the [PokeAPI](https://pokeapi.co/). It displays information such as the Pokémon’s name, image, type, height, and weight.


Features

- Search Pokémon by name or ID
- Fetches data from PokeAPI using `fetch()` and `async/await`
- Dynamically updates the DOM with Pokémon info
- Displays:
  - Pokémon name and image
  - Type(s)
  - Height and weight
- Handles invalid or empty searches with user-friendly error messages


Project Structure

- 3 pages, one HTML, one CSS and one JS
- simple HTML page with div created to display results
- simple CSS stylings to enhance the asthetic of the page
- JS file using async and await to interact with PokeAPI
